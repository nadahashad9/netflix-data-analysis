# Netflix Data Analysis

## Overview
A Big Data course project analyzing Netflix's global content library 
using PostgreSQL.

## Tools Used
- PostgreSQL (for data cleaning)
- Power BI (for visualization)

## What We Did
- Designed a relational database schema
- Cleaned and imported Netflix titles dataset
- Wrote SQL queries to extract insights
- Visualized results using Power BI
